# Longvity

Website of Longvity.

## Dependencies

* **ruby** - `2.0.x`
* **jekyll** - `1.0.x`

## Installation

Clone with [git(1)](http://help.github.com):

    $ git clone https://github.com/bodokaiser/longvity

Install `jekyll` with [gem(1)](http://rubygems.org):

    $ cd longvity && gem install jekyll

And call [jekyll(1)](http://jekyllrb.com)

    $ jekyll serve
